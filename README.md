Задача: Создать класс Phone, который описывает мобильный телефон.
Реализовано в папке src/phone

Требования:
Поля (приватные):

 	model (модель телефона) — строка

	batteryLevel (уровень заряда) — целое число от 0 до 100

Конструктор:

	Принимает модель и начальный уровень заряда.

	Если переданный уровень заряда меньше 0, установить 0. Если больше 100, установить 100.

Геттеры и сеттеры:

	getModel() — возвращает модель

	getBatteryLevel() — возвращает уровень заряда

	setBatteryLevel(int level) — устанавливает уровень заряда, но не даёт выйти за пределы 0–100

Методы:

	charge(int minutes) — увеличивает заряд на minutes (но не больше 100). Выводит сообщение: "Телефон [модель] заряжен до [уровень]%"

	use(int minutes) — уменьшает заряд на minutes (но не меньше 0). Выводит сообщение: "Использование [minutes] минут. Осталось заряда: [уровень]%"

	printInfo() — выводит информацию: "Модель: [модель], заряд: [уровень]%"

В методе main:

	Создай 2 телефона с разными моделями и начальным зарядом.

	Покажи работу всех методов.
